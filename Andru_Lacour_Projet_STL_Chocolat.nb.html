Qtests <- function(series, k, fitdf=0) {
  pvals <- apply(matrix(1:k), 1, FUN=function(l) {
    pval <- if (l<=fitdf) NA else Box.test(series, lag=l, type="Ljung-Box", fitdf=fitdf)$p.value
    return(c("lag"=l,"pval"=pval))
  })
  return(t(pvals))
}

Pour vrifier cela, on regarde si AR et le MA le plus grand de chaque modle est significatif. Pour cela, on regarde si la valeur absolue du coefficient divise par lcart-type est plus grande que 1.96 (ce qui correspond au seuil de 5%) : \(\frac{|Coeff|}{se(Coeff)} \ge 1.96\).

Concernant lARMA(1, 2), AR(1) et MA(2) sont tous les deux significatifs car \(\frac{0.9139}{0.0342} \simeq 26.72 \ge 1.96\) et \(\frac{0.6453}{0.0649} \simeq 9.94 \ge 1.96\).Donc le modle est bien ajust.

LARMA(0, 1) ne prend en compte que que MA(1) donc on va regarder ce coefficient seulement. Or, \(\frac{|-0.7322|}{0.0447} \simeq 16.38 \ge 1.96\). Donc lARMA(0, 1) est bien ajust.

adj_r2 <- function(model){
  ss_res <- sum(model$residuals^2)
  ss_tot <- sum(choco2[-c(1:max(p,q))]^2)
  p <- model$arma[1]
  q <- model$arma[2]
  n <- model$nobs-max(p,q)
  adj_r2 <- 1-(ss_res/(n-p-q-1))/(ss_tot/(n-1))
  return(adj_r2)
}
